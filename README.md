## Project Description
This project focuses on **Data Storytelling** using the Tips dataset. We analyze patterns in customer tipping behavior based on total bill, gender, day of the week, and group size to derive meaningful insights.
## Dataset Link
* **Dataset:** [Tips Dataset](https://raw.githubusercontent.com/mwaskom/seaborn-data/master/tips.csv)
## Output Screenshots

## Storytelling & Insights
* **Correlation:** We observed a positive relationship between the total bill and the tips provided.
* **Peak Times:** Data visualization shows higher tipping trends during weekends.
* **Customer Behavior:** Insights into how party size impacts service satisfaction.
## Instructions to Run the Project
1. Clone the repository.
2. Install necessary libraries: `pip install pandas seaborn matplotlib`
3. Run the notebook `Project2_Data_Storytelling.ipynb` in Jupyter or VS Code.
