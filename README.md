### **Project 2: Data Storytelling App**

# Project2_Data_Storytelling

## Project Description
This project focuses on **Data Storytelling** using the Tips dataset. We analyze patterns in customer tipping behavior based on total bill, gender, day of the week, and group size to derive meaningful insights through interactive visualizations.

## Dataset Link
* **Dataset Name:** Tips Dataset
* **Link:** https://raw.githubusercontent.com/mwaskom/seaborn-data/master/tips.csv
